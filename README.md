 ## 安装指南
 1. 下载本仓库中的`axure-chrome-extension.zip`文件。
 2. 解压下载的压缩包。
 3. 打开谷歌浏览器，进入`chrome://extensions/`页面。
 4. 开启右上角的“开发者模式”。
 5. 点击“加载已解压的扩展程序”，选择解压后的文件夹。
 6. 安装完成后，插件图标将出现在浏览器工具栏中。
